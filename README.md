# BMLP_Mei_Ratna

Repository ini berisi hasil pengerjaan submission **Belajar Machine Learning Pemula (Dicoding)** yang terdiri dari dua tahap, yaitu **Clustering** dan **Klasifikasi**.

Pada tahap clustering, data dikelompokkan menggunakan algoritma machine learning untuk menemukan pola dalam dataset. Hasil dari proses clustering kemudian digunakan sebagai target pada tahap klasifikasi.

Pada tahap klasifikasi, beberapa model machine learning seperti **Decision Tree** dan **Random Forest** digunakan untuk memprediksi target yang telah dihasilkan dari proses clustering. Model kemudian dievaluasi menggunakan metrik **accuracy, precision, recall, dan F1-score**, serta dilakukan **hyperparameter tuning** untuk meningkatkan performa model.

Melalui proyek ini saya belajar mengenai proses dasar dalam machine learning mulai dari **data preparation, modeling, evaluasi model, hingga penyimpanan model**.
